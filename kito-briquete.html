UI Kito Briquete

Este trabalho academico tinha como objetivo criar um site navegável usando as ferramentas aprendidas durante o curso.
Foi criado um site fictisio para compra de uma lenha ecológica chamada briquete. Este site conectaria tanto produtores e fazendeiros com seus respectivos compradores.
O site foi todo criado no FIGMA.
